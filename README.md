# MF_Player

MF_Player is a simple command-line media player application written in Python.

## Features

- Play media files directly from the command line.
- Lightweight and easy to use.

## Installation

**Dependencies:**
- [python-vlc](https://pypi.org/project/python-vlc/)

Install the required dependency using pip:

```bash
pip install python-vlc
```

## Usage

Run the script from your terminal:

```bash
python mf_player.py
```

You may be prompted to enter a file path or select media based on your implementation.
